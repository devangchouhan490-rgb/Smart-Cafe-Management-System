# Smart-Cafe-Management-System

Cafe Management System (C++)

A " console-based Cafe Management System " developed in **C++** using  Object-Oriented Programming (OOP) " concepts.  
This project simulates real-world cafe operations including **admin control, menu handling, combo offers, billing, GST calculation, UPI payment validation, and multi-table management**.



 **Features**

-> Admin Panel
- Secure admin login
- Set minimum bill amount discount
- Create custom combo offers
- Dynamic combo pricing

-> User / Customer Panel
- Add new customers dynamically
- Automatic table number assignment
- Supports maximum **15 tables**
- Category-wise menu:
  - Burgers
  - Pizza
  - Snacks
  - Main Course
  - Beverages
  - Desserts
- Quantity-based ordering
- Combo ordering
- Cancel order (table becomes free)

-> Billing System
- Table-wise bill generation
- Date, time, day & month printed on bill
- GST calculation
- Discount applied as per admin rules
- Payment modes:
  - Cash
  - Online (UPI format: `name@bank`)
- After billing, table is automatically freed
- Message displayed: **“Visit Again Sir
